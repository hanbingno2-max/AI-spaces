# 📖 AI Spaces 使用指南

## 简介

AI Spaces 是一个自动化的AI资讯聚合项目，每日自动获取AI新闻并推送到GitHub。

## 功能

- 📰 每日AI行业新闻自动获取
- 🤖 免费AI模型资讯汇总
- 📱 飞书即时推送
- 📂 GitHub版本管理

## 使用方法

### 1. 订阅飞书推送

配置完成后，每日7点会自动收到AI新闻推送。

### 2. 访问GitHub

- 仓库地址: https://github.com/hanbingno2-max/AI-spaces
- 每日新闻: `news/YYYY-MM-DD.md`
- 免费模型: `models/free-models.md`

### 3. 本地使用

```bash
# 克隆仓库
git clone https://github.com/hanbingno2-max/AI-spaces.git

# 查看今日新闻
cat news/$(date +%Y-%m-%d).md
```

## 自动任务

| 任务 | 时间 | 说明 |
|------|------|------|
| AI新闻获取 | 每日7:00 | 通过Tavily MCP获取 |
| 飞书推送 | 每日7:00 | 推送当日新闻 |
| GitHub提交 | 每日7:05 | 提交到仓库 |

## 常见问题

**Q: 如何修改推送时间？**
A: 修改cron任务配置。

**Q: 如何手动触发更新？**
A: 在OpenClaw中执行任务或等待自动执行。
