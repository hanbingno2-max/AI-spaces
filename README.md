# 🚀 AI Spaces

> AI行业资讯、每日AI新闻汇总、免费模型推荐

## 简介

本仓库用于记录每日AI行业新闻、免费模型资讯，由自动任务每日更新。

## 内容结构

```
AI-spaces/
├── README.md                 # 说明文档
├── docs/
│   └── 使用指南.md           # 使用方法
├── news/
│   ├── 2026-02-15.md        # 每日新闻
│   └── ...
├── models/
│   └── free-models.md       # 免费模型汇总
└── .github/
    └── workflows/
        └── daily-news.yml   # 自动更新工作流
```

## 每日自动更新

- 每日7:00 自动获取AI新闻
- 通过飞书推送当日新闻
- 自动提交到本仓库
